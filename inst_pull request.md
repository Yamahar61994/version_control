## Инскруция по pull request
1. Находим нужный нам репозиторий на Github
1. Нажмаем кнопку "**FORK**", и при этом создаётся копия на нашем аккунте Github.
1. С помошью команды "*git clone*" (ссылка на репозиторий на нашем аккунте) добавляем в папку репозиторий и переходим в него коиандой "cd".
1. Далее создаем инструкцию в новой ветке, `"pull_rec"`.
1. Командой "**git push**" отправляем на сервер Github.
